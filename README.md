# Simple Doctor Appointment Booking System

This is a beginner-friendly console-based doctor appointment booking application implemented in Python. It enables users to book appointments with doctors by entering their name, selecting a doctor from the list, and specifying the date and time.

## Features

- View the list of available doctors
- Book appointments by providing patient name, date, and time
- View all booked appointments
- Simple command-line interface, suitable for beginner programmers

## Purpose

This project is designed for college first-year students to learn fundamental programming concepts such as:

- User input and output handling
- Control flow with loops and conditionals
- Data storage using lists and dictionaries
- Structuring code with functions

It provides a practical example of developing a basic appointment system without the complexity of databases or web frameworks.
